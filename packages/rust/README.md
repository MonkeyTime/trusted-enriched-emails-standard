# Realtime Mail Rust SDK

Rust SDK for validation, canonicalization, and Ed25519 verification.

```bash
cargo test
```

This SDK is intended to become the best candidate for a shared security core and future WebAssembly bindings.

From the repository root:

```bash
npm.cmd run rust:check
```

On Windows, the MSVC Rust target also needs Visual Studio C++ build tools and a Windows SDK so Cargo can link build scripts and tests.
